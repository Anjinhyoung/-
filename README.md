### 유용한 소스코드인 거 같아서 이 레포지토리 들어올 때 마다 볼려는 코드(에라토네스의 체)
```python
n = int(input())
a = [True] * (n + 1)
m = int(n**0.5)

for i in range(2, m + 1):
    if a[i] == True:
        for j in range(i + i, n + 1, i):
            a[j] = False

print([i for i in range(2, n + 1) if a[i] == True])
```
* 위 코드는 입력된 수 이하에 모든 소수를 찾는 코드이다.
